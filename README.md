# ReX-v0
My custom Gym enviroment for the [Re_X](https://github.com/AXU9513/Re_X) project.

## Requirements
- Python 3
- [gym](https://gym.openai.com)
- [OpenCV](https://opencv.org) for Python

## Installation
```
pip install -e .
```

## Usage
In your Python script, do the following:
```
import gym
import gym_rex
env = gym.make('rex-v0')
```
